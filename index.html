Введение в git

Список полезных команд

touch - для создания пустого файла

ls - показать содержимое папки

cd - изменить(переместиться) папку

git --version - проверить версию или наличие git

git init - для инициализации репозитория

git clone - скопировать проэкт на компьютер

git status - проверка статуса файлов

git add - добавить файлы под контроль git
